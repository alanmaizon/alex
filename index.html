GitLab Duo Agent Platform Hackathon Project

Alex SDLC Copilot Agent

Alex is an AI agent flow that reacts to issue/MR events and automatically creates a structured SDLC triage artifact: risk level, test strategy, and release-readiness checklist.

Workflow automation (not chat-only)

  • Trigger: issue.opened or merge_request.opened
  • Action: execute scripts/alex_agent.py with the event payload
  • Result: generate .gitlab/agent-output/triage.md
  • Publish: attach output as an issue/MR note via workflow config

Quick run

Run locally:

python3 scripts/alex_agent.py --event-file examples/sample_event.json --output .gitlab/agent-output/triage.md
